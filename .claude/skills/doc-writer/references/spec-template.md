## Status 정의

| Status | 의미 |
|--------|------|
| `draft` | 초안. 구현 전 리뷰 단계 |
| `active` | 승인됨. 현재 구현의 동작 명세 |
| `deprecated` | 더 이상 유효하지 않음. 대체 Spec 또는 삭제 사유를 명시 |

새 Spec의 기본 상태는 `draft`다. 구현이 완료되고 테스트가 통과하면 `active`로 변경한다.
